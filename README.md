## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install packages.

```bash
pip install -r requirements.txt
```

## Preparar entorno
Para iniciar el codigo ocupamos
```bash
set FLASK_APP=main.py
```
o para linux
```bash
export FLASK_APP=main.py
```
## Iniciar servidor
```bash
flask run
```
